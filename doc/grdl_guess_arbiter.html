edit_distance/2

edit_distance(First, Second) -> any()

Calculates the editdistance from one string to another by calculating the number of character insertions, deletions, and moves it takes to change the first string to the second. The edit_distance function itself calls compare on the the parameter strings to compare the character changes from the first string to the second string.
